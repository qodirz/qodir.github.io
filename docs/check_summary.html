Check Summary

File name: YYYY-MM-DD_checksummary.csv

Description: The check summary is a rolled up check.

  • There should be a single row for every check, even if the entire check was comped or discounted.
  • If a check was split, there should still only be one check summary row.
  • Gross should be sum of line items pre-tax, pre-discount, pre-comp.
  • Net should be the sum of line items post-discount, post-comp, and pre-tax.
  • Discount should be the sum of all comps and discounts applied to check.
